Descripción del programa:

El programa diseñado es un sistema de búsqueda de países, que permite buscar, filtrar, ordenar y ver estadísticas de una selección de países. Entre los datos a mostrar para cada país, están: Nombre, Población, Superficie(En KM2) y Continente. Se le brindan varias opciones al usuario para que el pueda elegir entre las funciones:

Buscar país: Sirve para buscar un país en la lista, ingresando el nombre del mismo. El programa mostrará una lista de países que coinciden con los datos ingresados, mediante coincidencia total o parcial.

Filtrar países: Brinda opciones para filtrar países según el rango de población, continente o superficie.

Ordenar países: Brinda opciones para ordenar países según distintos ordenes: Alfabético, Por superficie y Por Población.

Mostrar estadísticas: Muestra algunas estadísticas, como por ejemplo: País con mayor y menor población, promedio de población y superficie de todos los países y cantidad de países en un continente.
